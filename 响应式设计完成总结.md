## 📱 移动端适配特性

### 断点设计

- **移动端**：≤ 768px
- **平板端**：769px - 1024px
- **桌面端**：≥ 1025px

### 响应式原则

1. **移动优先**：默认样式针对移动端，然后通过媒体查询适配大屏幕
2. **触控友好**：所有可点击元素最小尺寸44x44px
3. **横向滚动**：表格在移动端支持横向滚动而非隐藏列
4. **字体缩放**：移动端字体适当缩小，保持良好的可读性
5. **间距优化**：移动端间距减小，最大化内容显示空间

### 关键响应式功能

1. **抽屉菜单**：移动端使用抽屉菜单替代顶部导航
2. **表格横向滚动**：所有表格在移动端支持横向滚动
3. **表单优化**：移动端表单字段全宽，按钮全宽
4. **卡片布局**：移动端卡片单列显示，去除边距和阴影
5. **字体缩放**：移动端标题和文字适当缩小

## 🎨 响应式工具类

### 容器类
- `.navan-container` - 响应式容器（自动调整padding）

### 显示/隐藏类
- `.navan-mobile-only` - 仅在移动端显示
- `.navan-desktop-only` - 仅在桌面端显示

### 网格类
- `.navan-grid` - 响应式网格（1列→2列→3列→4列）

### 文本类
- `.navan-text-ellipsis` - 单行省略
- `.navan-text-ellipsis-2` - 两行省略
- `.navan-text-ellipsis-3` - 三行省略

### 表格类
- `.table-responsive-wrapper` - 表格响应式包装器（支持横向滚动）

## 📐 响应式间距系统

使用Navan CSS变量：
- `--navan-container-padding-mobile`：16px
- `--navan-container-padding-tablet`：24px
- `--navan-container-padding-desktop`：32px

## 🔧 技术实现

### CSS媒体查询

```css
/* 移动端 */
@media (max-width: 768px) {
  /* 移动端样式 */
}

/* 平板端 */
@media (max-width: 1024px) {
  /* 平板端样式 */
}

/* 桌面端 */
@media (min-width: 1025px) {
  /* 桌面端样式 */
}
```

### Ant Design响应式特性

- Table组件：`scroll={{ x: 'max-content' }}` 启用横向滚动
- Pagination组件：`responsive: true` 启用响应式分页
- Grid组件：使用Ant Design的栅格系统

### 移动端优化

1. **触控优化**：所有按钮和链接最小44px高度
2. **滚动优化**：`-webkit-overflow-scrolling: touch` 启用平滑滚动
3. **防止缩放**：`touch-action: manipulation` 防止双击缩放
4. **字体渲染**：移动端字体大小适当调整
