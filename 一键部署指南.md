## 🎯 方式1：通过Vercel网页一键部署（最简单）⭐

### 步骤1：访问Vercel并登录

1. **打开浏览器**
   - 访问：https://vercel.com

2. **登录账号**
   - 点击右上角 "Sign Up" 或 "Log In"
   - **推荐**：点击 "Continue with GitHub"
   - 使用您的GitHub账号（HMC1234）登录
   - 授权Vercel访问您的GitHub仓库

### 步骤2：导入项目

**方法1：从仓库列表选择**

1. 登录后，点击 "Add New..." → "Project"
2. 在仓库列表中找到：`HMC1234/-yqf-aibtrip-demo`
3. 点击 "Import"

**方法2：直接输入仓库地址**

1. 登录后，访问：https://vercel.com/new
2. 点击 "Import Git Repository"
3. 输入仓库地址：`https://github.com/HMC1234/-yqf-aibtrip-demo`
   - 或者直接输入：`HMC1234/-yqf-aibtrip-demo`
4. 点击 "Continue"

## 🎯 方式3：使用Vercel CLI（需要登录）

### 步骤1：登录Vercel CLI

在终端中执行：

```bash
vercel login
```

这会打开浏览器，完成登录。

### 步骤2：部署

登录后，执行：

```bash
vercel --prod
```

按提示操作即可。

**解决方案**：

- 检查是否有编译错误
- 确保所有依赖已安装

### 问题2：页面空白或404

**解决方案**：
- ✅ 已配置 `vercel.json`，会自动处理路由
- 如果还有问题，检查路由配置

### 问题3：Supabase连接失败

**解决方案**：
- 确认Supabase配置正确（已硬编码在代码中）
- 检查Supabase项目状态
