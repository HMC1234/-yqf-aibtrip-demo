## 移动端卡片设计规范

### 卡片结构
```
┌─────────────────────────────┐
│  Header（灰色背景）          │
│  单号/ID + 状态标签          │
├─────────────────────────────┤
│  Content（白色背景）         │
│  图标 + 标签：值             │
│  图标 + 标签：值             │
│  图标 + 标签：值             │
├─────────────────────────────┤
│  Footer（灰色背景）          │
│  [全宽按钮]                 │
└─────────────────────────────┘
```

### 间距规范
- 卡片间距：16px (var(--navan-space-4))
- 卡片内边距：16px (var(--navan-space-4))
- 信息行间距：12px (var(--navan-space-3))

### 字体规范
- 卡片标题（单号）：14px，加粗
- 标签文字：12px，灰色
- 值文字：14px，黑色
- 按钮文字：14px

### 触控规范
- 按钮高度：44px（符合触控标准）
- 卡片最小高度：120px
- 卡片可点击区域：整个卡片

### 视觉规范
- 卡片边框：1px solid var(--navan-gray-200)
- 卡片阴影：var(--navan-shadow-sm)
- 悬停阴影：var(--navan-shadow-md)
- 卡片圆角：var(--navan-radius-lg) (12px)

## 实现步骤

1. 在组件中添加移动端卡片列表（使用CSS控制显示/隐藏）
2. 移动端（≤768px）显示卡片列表，隐藏表格
3. 桌面端（>768px）显示表格，隐藏卡片列表
4. 使用相同的loading和empty状态
5. 卡片支持点击进入详情

## 响应式断点

- 移动端：≤ 768px - 显示卡片
- 桌面端：> 768px - 显示表格

## CSS类名规范

- `.mobile-cards-list` - 移动端卡片列表容器
- `.mobile-record-card` - 单个卡片
- `.mobile-card-header` - 卡片头部
- `.mobile-card-content` - 卡片内容
- `.mobile-card-footer` - 卡片底部
- `.mobile-card-title-row` - 标题行
- `.mobile-card-info-row` - 信息行
- `.mobile-card-icon` - 图标
- `.mobile-card-label` - 标签
- `.mobile-card-value` - 值
- `.desktop-table-wrapper` - 桌面端表格容器
