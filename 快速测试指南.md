# 快速测试指南

## 🎯 5分钟快速测试

### 步骤1：启动项目（1分钟）

**Windows**：
```bash
双击 一键启动.bat
```

**Mac/Linux**：
```bash
chmod +x 一键启动.sh && ./一键启动.sh
```

等待项目启动（自动打开浏览器 http://localhost:3000）

---

### 步骤2：登录系统（30秒）

1. 在登录页面输入：
   - **Email**: test@example.com
   - **Password**: 123456
2. 点击"登录"

---

### 步骤3：提交出差申请（1分钟）

1. 点击"提交新申请"按钮
2. 填写表单：
   - 出发地：北京
   - 目的地：上海
   - 日期：选择未来日期
   - 出差原因：测试AI推荐功能
   - 产品：勾选"机票"和"酒店"
3. 点击"提交申请"
4. 记住申请单号（例如：BTRQ+JD+...）

---

### 步骤4：更新申请单状态（1分钟）

**在Supabase Dashboard中**：

1. 打开 https://supabase.com/dashboard
2. 选择项目：YQFAIBTRIP
3. 打开 **Table Editor** → **travel_requests**
4. 找到刚创建的申请单
5. 将 `status` 字段改为 `approved`
6. 保存

**或者使用SQL**：
```sql
UPDATE travel_requests 
SET status = 'approved' 
WHERE request_no = '你的申请单号';
```

---

### 步骤5：测试AI推荐（2分钟）⭐

1. 返回应用，刷新页面
2. 在申请单列表中，点击"查看详情"
3. 点击"AI预订"按钮
4. **观看AI推荐生成过程**：
   - 步骤1：获取产品数据
   - 步骤2：核查差旅政策
   - 步骤3：生成推荐方案
   - 步骤4：完成
5. **查看推荐方案**：
   - 看到3-5个推荐方案卡片
   - 每个方案显示：关键字、推荐原因、价格
6. **查看方案详情**：
   - 点击"查看详情"
   - 查看完整的方案信息
