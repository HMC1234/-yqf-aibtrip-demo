```
首页
├── 预订管理
│   ├── 经典预订
│   │   ├── 机票预订
│   │   ├── 酒店预订
│   │   ├── 火车票预订
│   │   └── 用车预订
│   └── AI预订
│       ├── 智能对话预订
│       ├── AI推荐方案列表
│       └── 已审批出差申请列表
├── 出差申请管理
│   ├── 新的申请（提交出差申请）
│   └── 我的申请
│       ├── 已提交等待审批

│       ├── 审批完成

    └── 出行管理
```

## 🔄 主要页面流程关系

### 1. 出差申请流程
```
提交出差申请 → 我的申请（查看状态） → 审批完成 → 申请单详情
                                                      ├── 经典预订
                                                      └── AI预订 → AI推荐方案
```

### 2. AI预订流程
```
AI预订页面
├── 智能对话预订
│   ├── 生成出差申请单（BTRQ+AI+...）
│   └── 生成AI推荐方案（AIRD+AIC+...）
├── AI推荐方案列表
│   └── 推荐方案详情 → 一键预订（生成订单号：BK+AI+...）
└── 已审批出差申请列表
    └── 申请单详情 → AI预订 → AI推荐方案（AIRD+ZDT+...）
```

### 3. 经典预订流程
```
经典预订
├── 直接搜索预订（无需申请单）
└── 从申请单进入（带参数自动填充）
    └── 生成预订单（BK+TR+...）
```

### 4. AI推荐方案生成路径
```
AI推荐方案生成方式：
├── 方式1：已审批出差申请单 → AI预订 → 自动推荐（AIRD+ZDT+...）
│   └── 重新推荐（带关键字） → 个性化推荐（AIRD+GXH+...）
├── 方式2：智能对话 → 直接生成推荐（AIRD+AIC+...）
│   └── 重新推荐（带关键字） → 个性化推荐（AIRD+GXC+...）
└── 所有推荐方案 → 详情页 → 一键预订（生成订单号：BK+AI+...）
```

## 🎯 核心功能页面

### AI推荐方案详情页
```
AI推荐方案详情
├── 推荐单号
├── 核心关键字
├── 推荐原因说明
├── 推荐方案内容（3-5个方案卡片）
├── 重新推荐按钮
└── 一键预订按钮（生成订单号：BK+AI+年月日时分秒+五位顺序号）
```

### 出差申请单详情页
```
出差申请单详情
├── 申请单基本信息
├── 出发地/目的地
├── 出发日期/回程日期
├── 出差原因
├── 成本中心
├── 需要预订的产品（机票/酒店/火车票/用车）
├── 申请单状态

└── [AI预订] 按钮
```
