# Tag标签优化总结

**优化日期**：2025-11-21  
**优化范围**：所有页面中的Tag标签组件

## ✅ 已优化的Tag标签

### 优化前的问题
- Tag标签高度过高（约32-36px）
- 上下内边距过大（padding: 2px 12px）
- 行高过高，导致标签看起来过于"柱状"
- 视觉上不够紧凑美观

### 优化后的效果
- Tag标签高度：**22px**（固定高度）
- 上下内边距：**2px**
- 左右内边距：**8px**
- 行高：**18px**
- 字体大小：**12px**
- 圆角：**4px**
- 使用 `display: inline-flex` + `align-items: center` 实现完美垂直居中

---

## 📊 优化前后对比

### 优化前
```css
.ant-tag {
  padding: 2px 12px;  /* 左右内边距过大 */
  font-size: 14px;     /* 字体稍大 */
  height: auto;        /* 高度自适应，导致过高 */
  line-height: 1.5;    /* 行高过高 */
}
```
**效果**：标签看起来过高，像"柱子"一样

### 优化后
```css
.ant-tag {
  padding: 2px 8px;    /* 左右内边距减小 */
  height: 22px;        /* 固定高度，更紧凑 */
  line-height: 18px;   /* 行高优化 */
  font-size: 12px;     /* 字体稍小，更精致 */
  display: inline-flex;
  align-items: center;
  vertical-align: middle;
}
```
**效果**：标签紧凑美观，高度合适，视觉协调

## 🎨 统一的Tag标签规范

### 尺寸标准
- **高度**：22px（固定）
- **上下内边距**：2px
- **左右内边距**：8px
- **行高**：18px
- **字体大小**：12px
- **字体粗细**：500（medium）
- **圆角**：4px

### 对齐方式
- 使用 `display: inline-flex`
- 使用 `align-items: center` 垂直居中
- 使用 `justify-content: center` 水平居中
- 使用 `vertical-align: middle` 在表格中垂直居中
