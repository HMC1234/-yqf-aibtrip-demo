## 📊 优化前后对比

### 优化前
- 表头padding：`16px` 上下（`var(--navan-space-4)`）
- 表体padding：`20px` 上下（`var(--navan-space-5)`）
- 总行高：表头约56px，表体约60px
- **问题**：行高过高，显得稀疏不美观

### 优化后
- 表头padding：`10px` 上下，高度固定 `44px`
- 表体padding：`12px` 上下，高度固定 `48px`
- 总行高：表头44px，表体48px
- **改进**：行高紧凑，视觉更协调，信息密度更合理

## 🎨 统一的表格样式规范

### 表头样式
```css
.ant-table-thead > tr > th {
  padding: 10px 16px;
  height: 44px;
  font-size: 14px;
  font-weight: 600;
}
```

### 表体样式
```css
.ant-table-tbody > tr > td {
  padding: 12px 16px;
  height: 48px;
  vertical-align: middle;
  font-size: 16px;
}
```

### 移动端样式
```css
@media (max-width: 768px) {
  .ant-table-thead > tr > th,
  .ant-table-tbody > tr > td {
    padding: 8px 12px;
    height: auto;
  }
}
```
